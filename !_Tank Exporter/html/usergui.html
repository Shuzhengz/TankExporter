This is Tank Exporter.

When Tank Exporter is started for the first time, It will create data from the PKG files it needs to lookup and load tank files.
This takes a long time.. Be patient.. It's worth the wait.

The Main Window is the Render Window. This is where all visualization of the tank occurs.
In the bottom of this window the current frames per second is displayed and other information.

To the right is the Tank Tier lists. They are labeled by tier 1-10. Each of these tabs has a list of every tank by every nation in that tier's group.
The tier lists shows a small icon of the tanks type.. Heavy, Medium, Light, Tank Destroyer and Arty followed by its name in the PKG files.
The lower right window shows the tanks large icon when you mouse over one of the names on the tier lists. It also shows the name of the tank as seen in the game engine.
If you load a tank, this window will always return to the tank that's loaded.

To the left of the render window is a set of buttons that control different view settings.
Starting at the top these are.
1. Hide/Show the Chassis section.
2. Hide/Show the Hull section.
3. Hide/Show the Turret section.
4. Hide/Show the Gun section.

6. Hide/Show Terrain - Blank - Grid Pattern.
7. Hide/Show Polygon Edges.
8. Hide/Show Texture Selection Buttons.
9. Hide/Show Vertex Colors.

The Main Top Menu...
Under File/Settings, there are a number of settings to control what TE draws.
This is where you set the paths to the game folder (World of Tanks) and res_mods folder. The res_mods path must point to the current game version I.E. 1.0.0.1
You can test your model and wotmods creations by running WOT from "Start Game".
You can turn on and off shadows, bloom and other rendering items.
You can set the shadow quality and a few other things.
At the bottom of this menu are the debug settings for rendering, shadows and other tools I used while writing TE.
You can also edit the shaders though I don't recommend messing with them!
There are other items that will be covered in other areas of the help pages.

Under Export/Extract there are a few important items.
This menu item is disabled until a tank is loaded.
Extract to res_mods folder. This item is used to create the paths and files in the res_mods folder you will work. There will be much more about this under The Process Pages.
Export FBX is used to write out the loaded tank in FBX format. FBX is a widely used format for exchanging 3D model data.
View in res_mods folder will open explore and point it at the data in res_mods for the currently opened tank. If you have not extracted the data from the PKG files, A popup will tell let you know there is no data extracted for the loaded tank.
Remove Tank from res_mods. This will delete everything for the tank that resides in res_mods including the folders. DO NOT DELETE IMPORTANT WORK!
Screen Capture. This allows you to save a image of the current rendering.

Model Info opens a window that shows the vertex count and some other information. It also shows the information for any FBX that has been imported as well.

Camouflage opens the buttons on the bottom of the render window to pick different camos for the three seasons.
First.. Click season button followed by a camo pattern button. To remove a camo, click a different season button.
There are a few bugs with this process.. There are camos that are repeated and that are not available in the game.

Show Textures This turns color mapping on and off.. While off, the tank is rendered with no texturing.

GMM Toy opens a window that allows you to override the Gloss and Metalness of the GMM texture used in the PBR shader.

Lighting opens up the lighting control panel. On that panel, you can set the ambient, specular and overall brightness of the lighting.

Help. These help pages.

Decal Tool. This opens up the decal editing tool for adding, deleting and changing the decal layout in Tank Exporter. You can save your changes and they will be loaded every time TE starts up.
